# 🎬 MovieHub

O MovieHub é uma aplicação web desenvolvida em React que permite buscar filmes através de uma API externa e gerenciar uma lista de favoritos com persistência em nuvem.

O sistema foi desenvolvido como projeto acadêmico com foco em integração frontend + backend, consumo de API e implementação de CRUD.

---

## 🚀 Funcionalidades

- 🔍 Busca de filmes por nome (API TMDB)
- 🎞️ Exibição de filmes com imagem e avaliação
- ❤️ Adicionar filmes aos favoritos
- ❌ Remover filmes dos favoritos
- ☁️ Armazenamento dos favoritos em backend (Back4App)
- 🔄 Atualização dinâmica da interface
- 📱 Interface responsiva e estilizada

---

## 🛠️ Tecnologias Utilizadas

- React (Vite)
- JavaScript (ES6+)
- Axios
- React Router DOM
- CSS customizado
- API The Movie Database (TMDB)
- Back4App (Parse Server)

---

## 🧩 Arquitetura do Sistema

O projeto segue a seguinte arquitetura:

Frontend (React)
↓
Consumo de API externa (TMDB)
↓
Integração com Backend (Back4App - Parse)
↓
Persistência de dados (Favoritos)

---

## 🔄 CRUD Implementado

O sistema implementa um CRUD completo para a entidade "Favoritos":

- CREATE → adicionar filme aos favoritos
- READ → listar filmes favoritados
- DELETE → remover filme dos favoritos

Os dados são armazenados no Back4App, garantindo persistência em nuvem.
